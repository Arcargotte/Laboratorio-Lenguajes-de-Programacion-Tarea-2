Repositorio de propuestas de solución usando **Prolog** a los ejercicios planteados en la tarea 2 del laboratorio del curso **Lenguajes de Programación I** de la **Universidad Simón Bolívar** durante el trimestre **Septiembre-Diciembre 2025**.

## Descripción del programa

Este repositorio contiene dos asignaciones escritas en **Prolog**:  
- `prelaciones.pl`  
- `academico.pl`  

Ambos programas definen bases de conocimiento y reglas relacionadas con el contexto académico y las relaciones de prelación entre materias.

## Instrucciones de ejecución

### Instalación de Prolog en Windows
1. Visitar la página oficial de [SWI-Prolog](https://www.swi-prolog.org/).  
2. Descargar el instalador para Windows.  
3. Ejecutar el instalador y seguir las instrucciones predeterminadas.

### Abrir el intérprete de Prolog
Una vez instalado, abrir una terminal de comandos y ejecutar:
```bash
$ swipl
```

### Cargar las bases de conocimiento
Desde el intérprete, se pueden cargar los archivos .pl con los siguientes comando:

```
?- [prelaciones].
?- [academico].
```
